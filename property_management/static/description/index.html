Property Management Module

This module provides a comprehensive system to manage properties, tenants, leases, and rent payments in Odoo.

Features

  • Manage properties with types, prices, and statuses
  • Track tenant details and their leases
  • Create and manage lease contracts with automated status updates
  • Record and track rent payments
  • Generate QWeb PDF lease summary reports
  • Export lease data to Excel
  • View analytics with graph and kanban views
  • Automated lease expiry reminders via cron
